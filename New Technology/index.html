Composing with Components

In Vue, a component is essentially a Vue instance with pre-defined options. Registering a component in Vue is straightforward. You can see a component as a custom html-tag. 

Vue.component('todo-item', {
	template: '<li> This is a todo </li>'
})

This renders the same text over and over again, so we need to make the component so it accepts 'props'.

Vue.component('todo-item', {
	prop: ['todo'],
	template: '<li>{{ }}</li>' todo.text between two li's
})
